# 260711_geg2 - 포켓몬 탐험대 랜딩 페이지

Next.js, TypeScript, Tailwind CSS를 사용하여 구축한 포켓몬 소개 랜딩 페이지입니다.

## 주요 기능
* **포켓몬 탐험대 상단 메뉴**: 브랜드 로고 및 탐험 메뉴
* **반응형 히어로 섹션**: 데스크톱과 모바일 뷰에 최적화된 레이아웃
* **부드러운 애니메이션**: 페이드인 효과, 포켓몬 이미지 부유(Float) 효과, 버튼 인터랙션 호버 스케일
* **모던 UI 디자인**: 노란색과 파란색을 주요 색상으로 하고, 부드러운 배경 장식 요소를 배치한 아기자기한 디자인

## 기술 스택
* **Framework**: Next.js 16 (App Router)
* **Styling**: Tailwind CSS v4
* **Language**: TypeScript
* **Deployment**: Vercel에 최적화된 구조

## 실행 및 빌드 방법

### 개발 서버 실행
```bash
npm run dev
```

### 프로덕션 빌드
```bash
npm run build
```
